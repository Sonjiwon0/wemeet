# 🕒 WeMeet

**약속/모임 시간 조율 앱 (Next.js + TypeScript + Redux Toolkit)**  
사용자들이 가능한 시간대를 선택하고, 겹치는 시간대를 분석하여  
최적의 약속 시간을 추천해주는 웹앱입니다.

---

## 🔧 기술 스택

- **Next.js** (App Router)
- **TypeScript**
- **Redux Toolkit**
- **styled-components**
- **Vercel** 배포 예정

---

## 📌 주요 기능

- 약속 생성 (날짜/시간 범위 설정)
- 초대 링크 공유
- 사용자별 시간 선택 UI
- 겹치는 시간대 계산 → 추천 시간 표시
- 모바일 반응형 UI

---

## 📂 폴더 구조
├── app/          # 페이지 라우팅
├── features/     # 상태 관리 (Redux slices)
├── components/   # 공통 UI 컴포넌트
├── types/        # 타입 정의

## ✨ 개발 목표

- React + TS 실전 감각 향상
- Redux 상태 설계 패턴 학습
- 이직용 포트폴리오 프로젝트 구축
